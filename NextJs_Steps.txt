Parsing error : Cannot find module 'next/babel'
    Create file called .babelrc in your root directory and add this code:
        {
            "presets": ["next/babel"],
            "plugins": []
        }
    And in .eslintrc, replace the existing code with:
        {
            "extends": ["next/babel","next/core-web-vitals"]
        }
